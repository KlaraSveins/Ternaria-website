# Myndir sem þarf að setja hér

Allar myndir á nýju síðunni eru í dag tengdar beint frá gömlu Wix-síðunni (static.wixstatic.com). Þegar Wix verður lagt niður detta þessar myndir út. Settu réttu hrámyndaskrárnar inn í þessa möppu (`images/`) með nöfnunum hér fyrir neðan — þá er einfalt að skipta út `src=`-tilvísunum í HTML-skránum fyrir staðbundnar slóðir.

Dálkurinn "Núverandi (Wix)" er bara til viðmiðunar svo þið finnið réttu upprunalegu skrána í hrágagnamöppunni.
